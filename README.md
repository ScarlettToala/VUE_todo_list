
# Proyecto To-do List con Vue.js
Este proyecto es una aplicación de lista de tareas (To-do List) desarrollada con Vue.js. Permite a los usuarios agregar, modificar y eliminar tareas de manera sencilla. La aplicación está diseñada para ser fácil de usar y entender, ideal para quienes desean gestionar sus tareas diarias de forma rápida.

## Características
- Agregar nuevas tareas a la lista.
- Modificar tareas existentes.
- Eliminar tareas de la lista.
- Interfaz de usuario sencilla y amigable.

## Tecnologías Utilizadas
- Vue.js: Framework JavaScript progresivo para construir interfaces de usuario.
- HTML5 y CSS3: Para la estructura y el estilo de la aplicación.

## Instalación
1. Clona el repositorio:
```bash
git clone
```

2. Navega al directorio del proyecto:
```bash
cd To-do-VUE
```

3. Instala las dependencias:
```bash
npm install
```

4. Inicia la aplicación:
```bash
npm run serve
```

5. Abre tu navegador y visita `http://localhost:8080` para ver la aplicación en funcionamiento.
